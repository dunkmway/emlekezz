# Emlekezz (remember)

Text embedded note taking application. No folders or tags. Just notes.

## Features

- [ ] Store all notes
  - [ ] Title
  - [ ] Content (markdown)
  - [ ] Created Date
  - [ ] Modified Date
  - [ ] User
  - [ ]
  - [ ] Embedding
- [ ] View / Edit notes
  - [ ] Markdown / Richtext to markddown editor
  - [ ] Related Notes
  - [ ] Quick search (Ctrl / Command + K)
